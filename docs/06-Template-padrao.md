# Template padrão da aplicação

<span style="color:red">Pré-requisitos: <a href="02-Especificacao.md"> Especificação do projeto</a></span>, <a href="03-Metodologia.md"> Metodologia</a>, <a href="05-Projeto-interface.md"> Projeto de interface</a>

O template padrão do **Green Estoque** foi definido para garantir unidade visual, organização e facilidade de uso em todas as telas do sistema. A interface segue uma identidade visual inspirada na marca **Green Volt**, utilizando cores ligadas à tecnologia, energia limpa e confiança.

---

## Identidade visual

A identidade visual do sistema utiliza uma combinação de tons escuros, verdes e cores claras para criar uma interface moderna e profissional.

### Paleta de cores

| Elemento    | Cor sugerida | Uso                                            |
| ----------- | ------------ | ---------------------------------------------- |
| Azul escuro | `#111B4D`    | Fundo principal, cabeçalho e áreas de destaque |
| Verde menta | `#10B981`    | Botões, ícones, alertas positivos e destaques  |
| Branco      | `#FFFFFF`    | Fundo de cards, tabelas e formulários          |
| Cinza claro | `#F3F4F6`    | Fundo secundário e separadores                 |
| Vermelho    | `#DC2626`    | Alertas críticos e erros                       |
| Laranja     | `#F59E0B`    | Avisos e estoque baixo                         |

---

## Tipografia

A aplicação utiliza fontes sem serifa, priorizando boa leitura e aparência moderna. A tipografia deve ser simples, limpa e adequada para telas de dashboard, formulários e tabelas.

* **Títulos:** fonte em peso maior, destacando seções e telas;
* **Textos comuns:** fonte regular, com boa legibilidade;
* **Botões e menus:** texto objetivo e direto;
* **Tabelas:** fonte clara e espaçamento adequado.

---

## Layout padrão

As telas seguem uma estrutura visual padronizada:

* menu lateral para navegação;
* cabeçalho com título da página e ações principais;
* área central com cards, tabelas ou formulários;
* botões de ação destacados;
* feedback visual para erros, alertas e confirmações.

Essa organização facilita a navegação e permite que o usuário encontre rapidamente as funcionalidades principais do sistema.

---

## Componentes visuais

Os principais componentes utilizados na interface são:

| Componente  | Função                                               |
| ----------- | ---------------------------------------------------- |
| Cards       | Exibir indicadores e resumos do estoque              |
| Tabelas     | Listar produtos, fornecedores e movimentações        |
| Botões      | Executar ações como cadastrar, salvar, editar e sair |
| Formulários | Registrar produtos e movimentações                   |
| Alertas     | Informar estoque baixo, erros ou confirmações        |
| Ícones      | Facilitar a identificação visual das funcionalidades |

---

## Responsividade

O sistema foi planejado principalmente para uso em computadores e notebooks, considerando a rotina administrativa e operacional da empresa. Ainda assim, a interface deve se adaptar a diferentes tamanhos de tela, mantendo a navegação clara e os elementos organizados.

Em telas menores, os cards e tabelas devem se reorganizar para evitar perda de informação ou dificuldade de uso.

---

## Logo e marca

A marca **Green Volt** deve ser utilizada de forma discreta e padronizada no sistema, principalmente na tela de login, no menu lateral ou no cabeçalho. O uso da logo reforça a identidade da empresa e aproxima visualmente o sistema da organização atendida.

A aplicação deve evitar distorções, cortes ou alterações indevidas na logo. Sempre que possível, a marca deve aparecer sobre fundo azul escuro, seguindo o padrão visual da empresa.

---

## Tela de exemplo

A tela de exemplo escolhida para demonstrar o template padrão é o **Dashboard**, pois reúne os principais elementos visuais da aplicação: menu lateral, cards, indicadores, gráficos, alertas e atalhos de navegação.

![Template padrão - Dashboard](images/template-dashboard.png)

O dashboard utiliza cards para apresentar informações gerais do estoque, como quantidade de produtos cadastrados, itens com estoque baixo e movimentações recentes. Os botões e destaques utilizam verde menta, enquanto o azul escuro reforça a identidade da Green Volt.

---

## Considerações finais

O template padrão do Green Estoque busca manter consistência visual em todas as telas da aplicação. A padronização de cores, fontes, botões, cards e tabelas contribui para uma experiência mais clara, profissional e adequada à rotina da Green Volt.
